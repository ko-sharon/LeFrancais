## Exercice C
*notre/nos / votre/vos*

| Sentence | Answer | Note |
|---|---|---|
| «Nicole et Yvonne, — frères Jean et Henri, parlent-ils anglais ?» | **vos** | addressing Nicole et Yvonne (vous) → vos ; frères = plural |
| Nicole et Yvonne : «Non, — frères ne parlent pas anglais. Mais — père parle anglais.» | **nos; notre** | Nicole et Yvonne reply (nous) → nos (frères pl.) ; notre (père sing.) |
| «Quel âge — mère a-t-elle ?» Nicole et Yvonne : «— mère a trente-huit ans.» | **votre; Notre** | addressing them → votre ; they reply → Notre |
| «Est-elle plus âgée que — père ?» Nicole et Yvonne : «Non, elle est de deux ans plus jeune que — père.» | **votre; notre** | addressing → votre ; reply → notre |

## Exercice C
*demeure/demeures/demeurent / parle/parles/parlent*

| Sentence | Answer | Note |
|---|---|---|
| Les parents de Mme Lebrun — en Angleterre | **demeurent** | ils → -ent |
| Ils — anglais ; leur fille, Mme Lebrun, — anglais et français | **parlent; parle** | ils → parlent ; elle → parle |
| Mme Lebrun est Anglaise, mais elle — en France | **demeure** | elle → -e |
| «Jean et Henri, dans quelle ville — -vous ?» | **demeurez** | vous → -ez |
| Jean et Henri : «Nous — à Paris.» | **demeurons** | nous → -ons |
| «— -vous anglais ?» «Non, nous ne — pas anglais ; nous ne — que français.» | **Parlez; parlons; parlons** | vous → parlez ; nous → parlons (×2) |
| «Nicole, où — -tu en France ?» | **demeures** | tu → -es |
| Nicole : «Je — à Paris, dans la maison de mon père.» | **demeure** | je → -e |
| «Quelle langue — -tu ?» «Je — français.» | **parles; parle** | tu → parles ; je → parle |

## Exercice C
*je suis/tu es/il est/nous sommes/vous êtes/ils sont + j'ai/tu as/il a/nous avons/vous avez/ils ont*

| Sentence | Answer | Note |
|---|---|---|
| Les Français — les habitants de la France. Jean demeure en France ; il — Français. | **sont; est** | ils → sont ; il → est |
| «Jean, combien — -tu de sœurs ?» Jean : «J'— deux sœurs.» | **as; ai** | tu → as ; je → ai |
| «— -tu le fils de M. Duclos ?» «Oui, je — son fils.» | **Es; suis** | tu → es ; je → suis |
| «Combien de cousins — -vous, Nicole et Yvonne ?» «Nous — un cousin : Marcel.» | **avez; avons** | vous → avez ; nous → avons |
| Nicole — un chien, et Yvonne — un chat | **a; a** | elle → a (×2) |
| «Jean et Henri, — -vous les cousins de Marcel et de Monique ?» | **êtes** | vous → êtes |
| «Oui, nous — leurs cousins.» | **sommes** | nous → sommes |
| Marcel et Monique — deux cousins. | **ont** | ils → ont |

## 💡 Key Grammar Notes

- **notre/nos** = our ; **votre/vos** = your (vous form). Like all possessives, they agree with the noun owned: *notre père* (sing.) → *nos frères* (pl.).
- **avoir** is fully introduced here: j'ai, tu as, il/elle a, nous avons, vous avez, ils/elles ont.
- **être** is fully introduced here: je suis, tu es, il/elle est, nous sommes, vous êtes, ils/elles sont.
- **ne...pas** = not (total negation). **ne...que** = only (partial/restrictive negation). Both wrap the conjugated verb: *il ne parle pas* / *il ne parle que français*.
- **n'y a que** = there is/are only: *il n'y a que douze mois dans une année*.
- **en dehors de** = outside of. **à la campagne** = in the countryside. Contrast with **en ville** = in the city.
- **votre/vos** when addressing someone (vous), shifts to **notre/nos** when they reply (nous) — watch how this shifts in the dialogue throughout the chapter.
