# Guide de Prononciation Française

## The Big Idea First

French spelling is much more consistent than English spelling, but in the opposite direction: **many letters are silent, especially at the end of words.** Once you internalize a handful of rules, most words "click" automatically.

---

## Vowels — The Core Sounds

| Written | Sounds like | Example |
|---|---|---|
| **a** | "ah" | *chat* → shah |
| **e** (alone, mid-word) | short "uh" | *de* → duh |
| **é** | "ay" | *été* → ay-tay |
| **è / ê** | "eh" | *fête* → fet |
| **i / y** | "ee" | *ici* → ee-see |
| **o** | "oh" | *mot* → moh |
| **u** | no English equivalent — say "ee" but round your lips like "oo" | *du* → dyoo (roughly) |
| **ou** | "oo" | *vous* → voo |
| **eu / œu** | "uh" but with rounded lips | *peur* → puhr |

---

## Vowel Combinations (Digraphs)

| Written | Sounds like | Example |
|---|---|---|
| **ai / ei** | ay | *mais* → may, *seize* → sez |
| **au / eau** | oh | *beau* → boh, *aussi* → oh-see |
| **eu** | uh (rounded) | *deux* → duh |
| **oi** | wah | *moi* → mwah, *voir* → vwahr |
| **ou** | oo | *nous* → noo |
| **ui** | wee | *nuit* → nwee |
| **tion** | syon | *nation* → nah-syon |

---

## Nasal Vowels — Very French

When a vowel is followed by **m** or **n** and then another consonant (or end of word), the vowel becomes **nasal** — you push air through your nose. No real English equivalent, but here's the closest:

| Written | Sounds like | Example |
|---|---|---|
| **an / am / en / em** | "ahn" (like "on" in "bond" — British accent) | *dans* → dahn, *temps* → tahn |
| **in / im / ain / aim / ein** | "an" (like "an" in "and" — but nasal) | *vin* → van, *main* → man |
| **on / om** | "ohn" (like "on" but nasal) | *mon* → mohn, *bon* → bohn |
| **un / um** | like "an" above, slightly rounder | *un* → an |

The moment an **n** or **m** is **doubled**, or followed by a vowel, the nasality disappears:
- *bonne* → bun (not nasal — the **nn** breaks the rule)
- *animal* → ah-nee-mahl (not nasal — **n** is followed by vowel)

## Consonants — What's Different from English

Most consonants sound like English. The key exceptions:

| Letter | Rule | Sound |
|---|---|---|
| **r** | Guttural — made in the throat, not the tongue tip | Like a soft gargle |
| **g** before e/i | Like "zh" (the middle of "measure") | *genre* → zhohn-ruh |
| **g** before a/o/u | Like English "g" | *garçon* → gar-sohn |
| **c** before e/i | Like "s" | *ici* → ee-see |
| **c** before a/o/u | Like "k" | *café* → kah-fay |
| **ç** | Always "s" | *français* → frahn-say |
| **j** | Always "zh" | *je* → zhuh |
| **h** | Always silent | *Henri* → ahn-ree |
| **ch** | Always "sh" (never "ch" as in "cheese") | *chat* → shah |
| **qu** | Always "k" (the u is silent) | *que* → kuh |
| **th** | Just "t" (no English "th" sound) | *thé* → tay |

---

## Liaison — When Final Consonants Wake Up

There's one major exception to the "final consonants are silent" rule: **liaison**. When a word ending in a consonant is followed by a word starting with a vowel, the consonant **links** to that next word:

- *les enfants* → lay-zahn-fahn (the s of *les* links to *enfants*)
- *nous avons* → noo-zah-vohn
- *ils ont* → eel-zohn

This is why French speech sounds so smooth and flowing — words blend together.

## The Pairs That Confused You

### de vs des

- **de** → *duh* (the final **e** alone = "uh", nothing more)
- **des** → *day* (the combination **es** at word-end = "ay")

This is the key rule: **a final silent "s" changes the vowel sound before it.**

| Written ending | Sound | Examples |
|---|---|---|
| **-e** | uh | *de, le, que* |
| **-es** | ay | *des, les, mes* |
| **-er / -ez** | ay | *parler, mangez* |
| **-é** | ay | *été, café* |

So **le** = luh, **les** = lay. **mon** = mohn, **mes** = may.

---

### mai vs mais

- **mai** → *may* (May, the month)
- **mais** → also *may* — the final **s** is silent!

Wait — they sound the same? Yes! The difference is only in writing. This trips up English speakers because in English, final letters usually matter. In French, **final consonants are almost always silent.**

The letters most commonly silent at the end: **s, t, d, x, z, p**

| Word | Looks like | Sounds like |
|---|---|---|
| *mais* | mays? | may |
| *est* | est? | eh |
| *grand* | grand | grahn |
| *beaucoup* | boh-coop? | boh-coo |
| *deux* | duh-x? | duh |

## A Quick Reading Test

Try reading this line aloud before looking at the guide below it:

> *Les enfants parlent français.*

- **Les** → lay
- **enfants** → ahn-fahn (liaison: lay-zahn-fahn)
- **parlent** → par-luh (the **-ent** ending on verbs is always silent!)
- **français** → frahn-say

**The -ent verb ending being silent is one of the most important rules** — it's why *il parle* and *ils parlent* sound identical.
