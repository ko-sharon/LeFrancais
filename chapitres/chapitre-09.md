## 💡 Key Grammar Notes

- **Three tenses introduced side by side** — this is the heart of Chapter 9:
  - **L'imparfait** (imperfect) = was/were/used to. Signals: *hier, quand il était petit, avant*. Endings: -ait (sing.), -aient (pl.)
  - **Le présent** (present) = is/are/does. Signals: *maintenant, aujourd'hui, ce matin*.
  - **Le futur** (future) = will/going to. Signals: *demain, l'année prochaine, plus tard*. Endings: -ra (sing.), -ront (pl.)
- **Three verbs each across all three tenses:**
  - avoir: *avait / a / aura* — *avaient / ont / auront*
  - aller: *allait / va / ira* — *allaient / vont / iront*
  - être: *était / est / sera* — *étaient / sont / seront*
- **ne…plus** = no longer/not any more: *elle ne dort plus* = she is no longer sleeping. Different from *ne…pas* (not) and *ne…que* (only).
- **il y avait** = there was/there were (imparfait of *il y a*). **il y aura** = there will be (futur of *il y a*).
- **encore** = still / again. **pas encore** = not yet. **ne…plus** = no longer. These three form a set worth memorising together.
- **se laver** = to wash oneself (reflexive verb). *il se lave, ils se lavent* — the reflexive pronoun *se* always matches the subject.
